HABILIDADES

HTML

O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website.

Portanto, ele serve para dar significado e organizar as informações de uma página na web.

CSS

É usado para estilizar elementos escritos em uma linguagem de marcação como HTML.
O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página.
Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos.

JAVASCRIPT

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web

BOOTSTRAP

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo
